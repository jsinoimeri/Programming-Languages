To compile and execute this assignment automatically,
execute the batch file provided with this submition
by typing:

     ./compile_execute gram scan main
     
in the Linux terminal. This batch file assumes all 
the required files are in the same folder as it.

If error occurs trying to run the batch file, then
change the permissions on the file using:

     chmod 777 compile_execute
     
and rerun the batch file.
